# GODOT-Vehicle-Movement-Component

This is the development of a movement component designed to be attached to a vehicle, whether it is an aircraft vehicle or a wheeled vehicle

This component has not yet begun to be developed, at the moment it is simply an idea.

==================================================================

Se trata del desarrollo de un componente de movimiento diseñado para acoplarse a un vehículo, ya sea aéreo o con ruedas.

Este componente aún no se ha iniciado su desarrollo; por el momento, es simplemente una idea.
